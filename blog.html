Question-1: What are the different ways to select an element in the DOM?

There are 5 ways in which you can select elements in a DOM using selectors.

1. Finding HTML elements by id [getElementById()]
2. Finding HTML elements by tag name [ getElementsByTagName()]
3.Finding HTML elements by class name [(getElementsByClassName())]
4.Finding HTML elements by CSS selectors [querySelector()]
5. Finding HTML elements by HTML object collections [querySelectorAll()]

Question-3: What is event delegation in the DOM?

the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated when an event occurs on an Element it executes event handlers on that element and then propagates up to its parent and ancestors, so instead of adding events to its children , the event is declared to its parent node

Question-4: What is event bubbling in the DOM?

It is a process in DOM where an event is occured to a child node first and event runs and event bubbles to its parent node When an event occurs on an element, the event is first handled by the element itself. Then, the event propagates up to its ancestors (parent, grandparent, etc.), executing their respective event handlers if they exist. This continues until it reaches the document object.
